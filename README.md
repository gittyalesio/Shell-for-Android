# Shell for Android

A lightweight Android shell application written in Java.

## Features

- Execute shell commands using `/system/bin/sh`
- Built-in commands:
  - `cd`
  - `clear`
  - `exit`
- Current working directory
- Auto-scrolling terminal output
- Compatible with very old Android versions (minSdk 3)
- No AndroidX
- No AppCompat
- No Material Design
- Uses `android.app.Activity`

## Requirements

- Android Studio
- Java
