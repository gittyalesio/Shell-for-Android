# Changelog

All notable changes to this project will be documented in this file.

## [0.0.4] - 2026-07-17

### Added
- Shell command execution
- Built-in `cd`
- Built-in `clear`
- Built-in `exit`
- Enter key execution
- Auto-scrolling terminal
- Working directory support

### Changed
- Simplified interface
- Removed modifier bar
- Reduced MainActivity size to ~538 lines

### Fixed
- Touch issue determined to be caused by damaged HTC One M7 touchscreen, not the application.
